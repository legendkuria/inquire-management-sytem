How to run the Inquiry Management System (IMS) Project

1. Download the  zip file

2. Extract the file and copy IMS folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name inquiry

6. Import cms.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/IMS (frontend)

8. For admin panel

http://localhost/cms/admin  (admin panel)
